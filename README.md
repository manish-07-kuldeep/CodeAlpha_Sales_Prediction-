# Sales Prediction using Python

## Objective
Predict product sales using advertising expenditure on TV, Radio, and Newspaper.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Machine Learning Model
- Linear Regression

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Conclusion
The model predicts sales effectively and shows that TV advertising has the strongest impact on sales.# CodeAlpha_Sales_Prediction-
